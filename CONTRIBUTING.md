# Contributing

Contributions should identify an exact source version and a public, reproducible packet.

A proposed record should include:

1. versioned source identity;
2. exact statement label or location;
3. bounded result status;
4. replay or inspection route;
5. source and artifact hashes where available;
6. review state separated from internal replay;
7. rights and attribution routes.

Open an issue before submitting a new third-party corpus or statement body. Metadata links are preferred until the source-specific rights route is clear.
